# NyanMoji 🐱

> ✨ 由专属颜文字小精灵「喵符」(Nyafu) 为您倾情呈现！✨

喵符是一只头顶着`LED`猫爪小屏幕的猫娘，负责帮您把输入的拼音，变身成最可爱的颜文字魔法！(=^･ω･^=) 敲击键盘时，她可是在屏幕后面偷偷看着你的哦！

## 🐾 使用说明

- **快速召唤**：在任意文本框中输入 `/` + 拼音（比如输入 `/gaoxing`），立即弹出候选颜文字。
- **精准挑选**：使用键盘的上下方向键切换，或者按数字键 `1-9` 选定颜文字，按 `Esc` 键取消选择。
- **打开设置**：按下 `Alt+N`，或右键点击系统托盘的NyanMoji图标，可以呼出设置界面。在这里，你可以帮喵符换一身皮肤（主题切换）、管理她的“颜文字收藏夹”（词条管理），甚至导入/导出属于你自己的 `NyanMojiCode`！
- **浅/深色主题**：你也可以在设置里替换浅色或深色的主题。
- **开箱即用**：内置颜文字库，下载安装后无需任何额外配置，直接开打！

## 📦 安装与运行

### 方法一：直接运行 Python 脚本

1. 确保你的电脑里已安装 Python 3.8 或更高版本。
2. 克隆本仓库，或者直接下载源码包。
3. 安装依赖，在终端里敲下：
